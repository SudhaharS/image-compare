Visual Duplicate Detection Pilot:
This project demonstrates an Autonomous Enterprise solution for fraud prevention in warranty claims. By leveraging the Gemini Enterprise Agent Platform, we move beyond text-based audits to "Visual Fingerprinting."

Core Capabilities:
Multimodal Embedding: Converts raw images of auto parts into 1408-dimension semantic vectors.

Visual Similarity: Uses Cosine Similarity to detect if a dealer is re-submitting a previously paid claim photo, even if the photo is cropped, rotated, or renamed.

Cloud Native: Fully serverless architecture running on GCP Cloud Run with automatic scaling.
